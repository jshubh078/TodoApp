# Todo App Project

## Perquisites (Development):
| Module | Version |
| --- | --- |
| Node | 14.16.0 |
| Npm | 6.13.0 |
| sequelize | ^6.3.2 |

##### Take Clone of project
> git clone -b git_url  folder_name

##### Rename configSample.js to configs.js
> cd configs
> mv configSample.js configs.js

##### Set Db Url in env.dev file in db variable 

> db = your local postgresql data base url

##### Install node modules

> npm install

##### Swagger Docs Url and username & password

> url http://localhost:5010/docs/

> username :- todoApp 
> password :- todoApp
